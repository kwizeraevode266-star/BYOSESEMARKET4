Quick Integration (3 Steps)

Step 1: Include CSS

Add this line to the <head> of your HTML file:

<link rel="stylesheet" href="css/categories.css">

Step 2: Add Categories Component

Include the categories modal HTML in your <body>:

<!-- Categories Component --> <!-- Copy entire content from components/categories.html -->

Or use dynamic loading:

<div id="categories-container"></div> <script> fetch('components/categories.html') .then(r => r.text()) .then(html => { document.getElementById('categories-container').innerHTML = html; }); </script>

Step 3: Include JavaScript

Add this script before closing </body>:

<script src="js/categories.js"></script>
✅ Done! The Categories section is now ready. Tap the "Categories" button in your mobile navigation to test it.

Customization Guide

Add or Remove Categories

Edit the CATEGORIES array in js/categories.js:

const CATEGORIES = [ { id: 'shoes', name: 'Shoes', icon: 'fa-solid fa-shoe-prints', color: '#2563eb', path: 'shop.html?category=shoes', count: 245 }, // Add more categories here ];

Change Primary Colors

Modify CSS variables in css/categories.css:

:root { --categories-primary: #2563eb; /* Main brand color */ --categories-secondary: #1e40af; /* Hover/darker shade */ --categories-light-bg: #f8fafc; /* Card background */ --categories-text-primary: #0f172a; /* Main text */ --categories-text-secondary: #64748b; /* Secondary text */ }

Adjust Grid Layout

Change number of columns in css/categories.css:

/* Mobile: 2 columns */ .categories-grid { grid-template-columns: repeat(2, 1fr); } /* Tablet: 3 columns */ @media (min-width: 480px) { .categories-grid { grid-template-columns: repeat(3, 1fr); } }

Shop Page Integration

To handle category filtering in your shop page, add this code to shop.js:

// Get category from URL parameter const params = new URLSearchParams(window.location.search); const selectedCategory = params.get('category'); // Filter products based on category function filterProductsByCategory(categoryId) { const products = document.querySelectorAll('[data-category]'); products.forEach(product => { if (categoryId && product.dataset.category === categoryId) { product.style.display = 'block'; } else if (!categoryId) { product.style.display = 'block'; } else { product.style.display = 'none'; } }); } // Initialize document.addEventListener('DOMContentLoaded', () => { if (selectedCategory) { filterProductsByCategory(selectedCategory); // Optional: show category name console.log('Showing products for:', selectedCategory); } });
Note: Ensure your product HTML has data-category="shoes" etc.
