📋 Integration Steps

Step 1: Include CSS and JavaScript

Add these to your HTML file:

<!-- In <head> --> <link rel="stylesheet" href="css/categories.css"> <!-- Before closing </body> --> <script src="js/categories.js"></script>

Step 2: Add Categories Component

Include the categories modal HTML anywhere in your body:

<!-- Before closing </body> --> <!-- Categories Component --> <!-- [Paste content from components/categories.html here] -->

Step 3: Mobile Navigation

Ensure your mobile navigation has a Categories button:

<a href="#" class="mobile-bottom-nav__item" data-nav="categories"> <span class="mobile-bottom-nav__icon"> <i class="fa-solid fa-list"></i> </span> <span class="mobile-bottom-nav__label">Categories</span> </a>

The JavaScript will automatically detect and connect this button to the categories modal.

🎨 Customization

Add/Remove Categories

Edit the CATEGORIES array in js/categories.js:

const CATEGORIES = [ { id: 'shoes', name: 'Shoes', icon: 'fa-solid fa-shoe-prints', color: '#2563eb', path: 'shop.html?category=shoes', count: 245 }, // Add more categories here... ];

Change Colors

Modify CSS variables in css/categories.css:

:root { --categories-primary: #2563eb; /* Main brand color */ --categories-secondary: #1e40af; /* Hover state */ --categories-light-bg: #f8fafc; /* Card background */ --categories-text-primary: #0f172a; /* Primary text */ /* ... more variables ... */ }

📚 JavaScript API

The Categories Manager is automatically initialized. Access it via:

// Get the instance const categoriesManager = window.categoriesInstance; // Available methods: categoriesManager.open(); // Open modal categoriesManager.close(); // Close modal categoriesManager.toggle(); // Toggle open/close categoriesManager.getCategory(id); // Get category by ID categoriesManager.getAllCategories(); // Get all categories categoriesManager.updateCategories(array); // Update categories

Example: window.categoriesInstance.open();
