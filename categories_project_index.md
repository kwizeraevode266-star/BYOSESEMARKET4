## 🔧 Customization Examples

### Change Color Theme (30 seconds)
Edit `css/categories.css`:
```css
:root {
  --categories-primary: #FF6B6B; /* Your brand color */
}
```

### Add New Category (1 minute)
Edit `js/categories.js`:
```javascript
{
  id: 'new-category',
  name: 'New Category',
  icon: 'fa-solid fa-star',
  color: '#FF6B6B',
  path: 'shop.html?category=new-category',
  count: 0
}
```

### Change Grid Layout (30 seconds)
Edit `css/categories.css`:
```css
.categories-grid {
  grid-template-columns: repeat(4, 1fr); /* 4 columns instead of 2-3 */
}
```

See [CATEGORIES_INTEGRATION_GUIDE.md](CATEGORIES_INTEGRATION_GUIDE.md) for more customization options.
