💡 JAVASCRIPT API

window.KCart.add(item)
  └─ Add item to cart

window.KCart.remove(productId)
  └─ Remove item from cart

window.KCart.updateQty(productId, quantity)
  └─ Update item quantity

window.KCart.clear()
  └─ Clear entire cart

window.KCart.render()
  └─ Render cart UI

window.KCart.renderCount()
  └─ Update badge count

window.KCart.openWhatsAppOrder(customMessage)
  └─ Open WhatsApp with order
